# Конвертер характеристик кондиционеров

Скрипт для автоматического объединения прайс-листов и каталогов климатической техники. Он сопоставляет разрозненные названия характеристик (синонимы) и приводит их к единому стандарту.

## Требования

- **Python 3.8+**
- Библиотеки: `pandas`, `openpyxl`

## Установка и запуск

1. **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/LianetHub/phyton-product-sorter
    cd phyton-product-sorter
    ```

2. **Установите зависимости:**

    ```bash
    pip install pandas openpyxl
    ```

3. **Подготовьте данные:**
   Поместите исходные `.xlsx` файлы в папку `data/`.

4. **Запустите скрипт:**
    ```bash
    python script.py
    ```

## Структура проекта

```text
phyton-product-sorter/
├── script.py              # Основной код (логика сопоставления и обработки)
├── README.md              # Документация проекта
├── data/                  # Директория для входящих Excel-файлов
│   ├── vendor_1.xlsx      # Пример входных данных
│   └── vendor_2.xlsx
└── merged_catalog.xlsx    # Итоговый унифицированный файл (генерируется скриптом)
```
