# Конвертер характеристик кондиционеров

Скрипт для автоматического объединения прайс-листов и каталогов климатической техники. Он сопоставляет разрозненные названия характеристик (синонимы) и приводит их к единому стандарту.

## Возможности

- **Умное сопоставление:** объединяет колонки с разными названиями (например, "Бренд" и "Производитель") в одну.
- **Сбор фильтров:** автоматически сканирует все колонки, содержащие упоминания фильтров тонкой очистки, и собирает их в единый список для каждого товара.
- **Логирование:** подробный вывод всех этапов работы в консоль.

## Требования

- Python 3.8 или выше
- Библиотеки: `pandas`, `openpyxl`

## Установка

1. Клонируйте репозиторий:

    ```bash
    git clone <url_вашего_репозитория>
    cd <название_папки>

    ```

2. Установите необходимые зависимости:

Bash
pip install pandas openpyxl

Использование
Поместите исходные файлы .xlsx в папку data/.

Запустите скрипт:

Bash
python script.py
Результат будет сохранен в файл merged_catalog.xlsx.

Структура проекта
script.py — основной файл скрипта.

data/ — папка для входящих файлов (создается автоматически, если её нет).

merged_catalog.xlsx — итоговый отчет.
